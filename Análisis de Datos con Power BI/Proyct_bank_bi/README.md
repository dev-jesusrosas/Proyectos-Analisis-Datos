📊 Proyecto: Análisis de Datos Bancarios con Power BI
Este proyecto presenta una solución integral de inteligencia de negocios para el sector financiero, transformando datos complejos en un tablero de control estratégico.
🧠 Arquitectura y Modelado de Datos
El núcleo de este informe se basa en un Modelo Dimensional (Esquema en Estrella), garantizando eficiencia y escalabilidad:
Tablas de Hechos (Fact Tables): Centralización de métricas de negocio y registros transaccionales bancarios.
Tablas Maestras (Dimensions): Catálogos normalizados de clientes y productos para segmentación precisa.
Inteligencia de Tiempo: Implementación de tablas de tiempo para análisis comparativos (YTD, YoY) y tendencias históricas.
🚀 Habilidades Técnicas Aplicadas
DAX Avanzado: Creación de medidas para cálculo de indicadores clave (KPIs).
Power Query: Limpieza, transformación y estructuración de los datos del banco.
Modelado de Datos: Diseño de relaciones entre tablas maestras y de hechos para un rendimiento óptimo.
📂 Contenido de la Carpeta
Tablas maestras, hechos y tiempo.pbix: Archivo principal con el dashboard y modelo.
datos_personales_banco.xlsx: Fuente de datos utilizada para el análisis.
